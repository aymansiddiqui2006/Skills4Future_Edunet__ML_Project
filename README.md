# ☀️ Solar Intelligence System

## 🚀 Overview

An AI-powered system that predicts solar energy generation using weather data and provides insights, optimization strategies, and business impact analysis.

---

## 🎯 Problem Statement

Solar energy output depends heavily on environmental conditions.
This project builds a machine learning system to:

* Predict solar power generation
* Analyze influencing factors
* Provide optimization insights

---

## 🧠 Features

* ⚡ Solar Power Prediction (ML Model)
* 📊 Interactive Dashboard (Streamlit)
* 📈 24-Hour Forecast Visualization
* 🧠 Feature Importance Analysis
* 🔮 Scenario Simulation
* 💰 Revenue Estimation
* 🤖 AI-based Recommendations

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* Plotly

---

## 📂 Dataset

* Kaggle Solar Power Generation Dataset
* Includes:

  * Power Generation Data
  * Weather Sensor Data

---

## ⚙️ How to Run

### 1. Clone repo

```bash
git clone https://github.com/your-username/solar-intelligence-system.git
cd solar-intelligence-system
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run app

```bash
python -m streamlit run app.py
```
